## 🚀 How to Use

### **Quick Start (3 Steps):**

**Step 1: Download Template**
```
1. Click "📤 Data Management" in sidebar
2. Click "📋 Template" tab
3. Click "📥 Download Template (CSV)" button
4. Save to your computer
```

**Step 2: Prepare Your Data**
```
1. Open template in Excel
2. Fill with your loan data
3. Match format exactly
4. Save as CSV
```

**Step 3: Upload & Use**
```
1. Go to "📥 Import Data" tab
2. Upload your CSV file
3. System validates automatically
4. Review preview
5. Data ready for use!
```

## 🛠️ Technical Details

### Session State Management
- Uploaded data stored in `st.session_state`
- Persists during session
- Can be reset to default anytime

### Data Processing
- Missing values auto-filled
- Categorical values encoded
- Numeric ranges validated
- Quality checks performed

### File Support
- **Format:** CSV (Comma-Separated Values)
- **Size:** Up to 100MB+
- **Rows:** Supports 10,000+ records
- **Encoding:** UTF-8 supported

## 💡 Pro Tips

### ✅ Best Practices
- Download template first
- Match format exactly
- Test with 10 rows first
- Review data preview
- Keep original backup
- Document any changes

### ❌ Common Mistakes to Avoid
- Changing column names
- Using different text formats
- Including $ in numbers
- Not saving as CSV
- Uploading corrupted files
- Skipping validation

---

## 🆘 Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| "Missing columns" | Download template and match column names exactly |
| "File won't upload" | Ensure it's CSV format, not Excel |
| "Data shows as rejected" | Check credit scores (300-850) and income ranges |
| "Format error" | Review template for correct categorical values |
| "No data preview" | Check file has header row and is not empty |
